# Data Directory

Place the `Flight_Price_Dataset_of_Bangladesh.csv` file in this directory.

## Expected CSV Columns

- Airline
- Source
- Source Name
- Destination
- Destination Name
- Departure Date & Time
- Arrival Date & Time
- Duration (hrs)
- Stopovers
- Aircraft Type
- Class
- Booking Source
- Base Fare (BDT)
- Tax & Surcharge (BDT)
- Total Fare (BDT)
- Seasonality
- Days Before Departure

## Download Instructions

1. Download from Kaggle: [Flight Price Dataset of Bangladesh](https://www.kaggle.com/datasets/)
2. Place the CSV file in this directory
3. Ensure the filename is exactly: `Flight_Price_Dataset_of_Bangladesh.csv`
